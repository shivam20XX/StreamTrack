StreamTrack is a personal media tracking application that helps you organize, rate, and discover movies and TV shows wihout any login and tracking stuff. Keep track of everything you've watched and build your perfect watchlist.

## Features

- Track movies and TV shows you've watched
- View detailed information from TMDB (The Movie Database)
- Maintain a personal watchlist
- Rate and review media
- Search and discover new content
- Get personalized recommendations based on your viewing habits


- Keep a viewing history log
## Tech Stack

- **Python** - Backend logic
- **Streamlit** - Web framework for interactive UI
- **TMDB API** - Movie and TV show data

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Get a TMDB API key from [themoviedb.org](https://www.themoviedb.org/settings/api)
4. Add your API key to the configuration ( in .env file )
5. Run the app: `streamlit run app.py`

## Usage

Launch the app and search for movies or TV shows to add to your personal collection. Track viewing history and manage your watchlist directly from the interface.
